# Project Intro HyperFrames Video

This HyperFrames project renders a Chinese product-introduction video for
Personal OS + Personal Wiki.

It is intentionally about the product loop, not release packages or
cross-platform installers:

```text
messy input -> Wiki memory -> task -> agent claim -> evidence -> review
```

Render:

```bash
npm run check
npm run render -- --output ../../personal-os-wiki-project-intro.zh-CN.mp4 --quality standard
```
