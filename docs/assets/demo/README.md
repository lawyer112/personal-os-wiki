# README Demo Assets

These assets are safe public demo media generated from fake data.

The demo flow is:

```text
Telegram-style link intake -> Personal OS Inbox -> Task + Wiki link
  -> Personal Wiki note -> Telegram-ready reminder payload
```

Files:

- `personal-os-wiki-readme-demo.gif` is the inline README preview.
- `personal-os-wiki-readme-demo.mp4` is the higher-quality linked video.
- `01-telegram-input.png` through `05-telegram-reminder.png` are source frames.

Do not replace these with real inbox items, real task history, private Wiki
notes, tokens, browser chrome screenshots, private network maps, or deployment
paths. Public demo media should use fake data only.
